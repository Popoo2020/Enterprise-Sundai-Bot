Integritetsmeddelande

Detta meddelande förklarar hur SundAI hanterar uppgifter som skickas in via webbplatsen.

Personuppgiftsansvar

SundAI drivs av Riimon Holdings LTD. Företaget ansvarar för personuppgifter som skickas in via webbplatsen.

Uppgifter vi tar emot

Kontaktformuläret kan samla in namn, e-postadress, organisation och innehållet i ditt meddelande. Tekniska hostingloggar kan också behandlas för säkerhet och tillförlitlighet.

Ändamål

Uppgifterna används för att besvara förfrågningar, förbereda relevant uppföljning och skydda webbplatsen mot missbruk. Vi säljer inte uppgifter från kontaktformuläret.

Leverantörer och överföringar

Webbplatsen hostas hos Cloudflare. När e-postleverans är konfigurerad överförs formulärmeddelanden via Resend. Leverantörerna behandlar uppgifter enligt sina dataskyddsvillkor och tillämpliga skyddsåtgärder.

Lagring

Förfrågningar sparas endast så länge det rimligen behövs för kommunikation, dokumentation och tillämpliga rättsliga skyldigheter.

Dina val

Du kan begära tillgång, rättelse eller radering av uppgifter kopplade till din förfrågan genom att skicka ett nytt meddelande via kontaktformuläret och identifiera den ursprungliga förfrågan.

Uppdateringar

Meddelandet kan uppdateras när webbplatsen, leverantörer eller behandlingar förändras. Senast uppdaterat: 23 juli 2026.
