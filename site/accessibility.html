Accessibility statement

SundAI aims to provide a website that is clear, keyboard-accessible and usable across devices.

Current measures

The site uses semantic headings, visible keyboard focus, labelled form fields, sufficient contrast, responsive layouts, reduced-motion support and a skip-to-content link.

Known limitations

The animated visual elements are decorative and do not communicate essential information. Third-party or future embedded content may have separate accessibility limitations.

Feedback

Please use the contact form to report an accessibility problem. Include the page, device and a short description so the issue can be investigated.
